Version 1.0.beta

The EA:Ragnar√∂k Adepta Sororitas list is based on the E-UK version of the list, but with some siginficant changes to force structure, stats and special rules.

Faithful: The Faithful rule has been revised succeed automatically, but only if the formation carries out an Engage action that leads to an Assault. This makes this rule less abusable, in particular it prevents Fearless scouts from marching up and blocking off entire sections of the battlefield. In addition, formations that use the Faithful rule counts as having a character with 'Inspiring' in the formation.

The Exorcists have been split into two versions (one direct fire (Prioris) and one equipped with a Whirlwind Missile Launcher (Sanctorum)). The Exorcist Prioris have longer range but worse AT. It also have a 5+ Armour, since it is based on the Rhino.

Battle Sisters: This formation have had their Heavy Flamers replaced with Heavy Bolters, as we found them to be somewhat too strong with Heavy Flamers. Battle Sisters Mission is also the sole Core formation in the Army, but is more flexible with options to replace Battle Sisters with Exorcist Prioris or to ride in a Sororitas Stormlord.

No mixed Sister formations: In order to reduce confusion for players (there are quite a few types of "Sisters" in the list), each type of Sister unit now has its own type of formation, with the Celestial Sisters merging with the Immolator formation to create a quite expensive but very nasty Immolation formation.

Machine Spirits: Certain units which are basically servitors and are regarded as cannon fodder have been given the Machine Spirit rule, and are also Fearless and Expendable. These units are now upgrades to the Redemptionist formation.

Imperial Navy: The Lightning Strike Fighters have been replaced with the Lightning Interceptor, and the ground attack role been taken over by the Marauder Destroyer, which also replaces the Marauder Squadron. Since the list now contains Exorcist Sanctorums these are less needed.

Additional nerfs, boosts & tweaks
  • The Inquisitor formation can now contain a Supreme Commander.
  • A formation of four Hellhounds have been added to the list.
  • Rhinos are no longer free, but must be bought as an upgrade. This makes it more attractive to take other kinds of transport.
  • The Repentia Sisters formation is different, and now contains the new Mortifier type unit.
