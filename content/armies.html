Armies

Most EA:Ragnarök list are based on lists developed by the NetEA, Epic-UK, and the F-ERC army list development groups. We have done a few general changes across all EA:Ragnarök lists as well as various tweaks, nerfs and boosts. Each list contains a description of the list origin and the changes we made from the original.

Each list includes an Army Forge list builder link, which can also can be accessed here.
